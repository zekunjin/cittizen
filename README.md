# Ctizen

🙋🏻‍♂️ Smart CLI builder framework powered by citty.

## Usage

Install package:

```bash
# npm
npm install citty && npm install ctizen -D

# yarn
yarn add citty && yarn add ctizen -D

# pnpm
pnpm add citty && pnpm add ctizen -D
```

Update package.json:

```json
{
  "script": {
    "build": "cti build",
    "dev": "cti dev"
  }
}
```

## Configuration

In ctizen.config.ts:

```ts
export default {
  tsup: {}
}
```
